How to contribute

  1. Clone the repo
  2. Create a new branch
  3. Add your route in views.py
  4. Add any python, JS, HTML, or CSS code.
    Make sure it's named the same as your route(organizational purposes only)
    * JS and CSS files go in static
    * HTML templates go in templates
    * Python files go in the root directory
  5. Commit the changes
  6. Submit a pull request
